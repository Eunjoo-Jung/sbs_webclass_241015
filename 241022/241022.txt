CSS : Cascading Style Sheet

전체선택자 : Wild card Selector : *

태그선택자 : 특정 태그 

아이디선택자 : html > 태그 > id => #

클래스선택자 : html > 태그 > class => .

후손선택자 : 조부모 <-> 손주

자손선택자 : 부모 <-> 자식 (ex)ul, li) -> 특정태그의 바로 밑에 오면 자손선택자

속성선택자 : html 태그 중에서 속성 및 속성값을 가지고 있어야 함!

동위선택자 : + // - : 같은 라인이 존재하는 형제 요소 찾아내고자 할 때

일반구조선택자 : 부모 밑에 모든 자식요소의 태그가 동일한 경우

형태구조선택자 : 부모 밑에 자식요소의 태그가 상이한 경우
